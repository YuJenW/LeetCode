# 151. Reverse Words in a String 解題說明

## 1. 題目說明

給定一個字串：

```text
s
```

請將字串中的**單字順序反轉**，並回傳新的字串。

---

## 2. 題目要求

題目需要同時滿足以下條件：

```text
1. 單字順序要反轉
2. 輸出中的單字之間只能有一個空格
3. 不可以有開頭空格
4. 不可以有結尾空格
5. 原字串中若有多個連續空格，輸出時要壓縮成一個空格
```

---

## 3. 單字的定義

題目中：

```text
A word is defined as a sequence of non-space characters.
```

意思是：

> 一個單字就是一段連續的非空白字元。

例如：

```text
"the sky is blue"
```

可以拆成：

```text
the
sky
is
blue
```

---

## 4. 題目範例

### Example 1

```text
Input:
s = "the sky is blue"

Output:
"blue is sky the"
```

---

### Example 2

```text
Input:
s = "  hello world  "

Output:
"world hello"
```

原字串有：

```text
開頭空格與結尾空格
```

輸出時都需要移除。

---

### Example 3

```text
Input:
s = "a good   example"

Output:
"example good a"
```

原本：

```text
good 與 example 中間有多個空格
```

輸出時只保留：

```text
一個空格
```

---

## 5. 這題的核心想法

這題要反轉的是：

```text
單字的順序
```

而不是：

```text
整個字串中的每一個字元
```

---

### 例如

```text
"the sky is blue"
```

不是反轉成：

```text
"eulb si yks eht"
```

而是：

```text
"blue is sky the"
```

---

## 6. 解法策略：從右往左掃描

因為題目要把單字順序反過來，所以可以直接：

```text
從字串最右邊開始往左掃描
```

這樣遇到的單字順序剛好就是答案需要的順序。

---

### 例子

```text
s = "the sky is blue"
```

從右往左看，會依序遇到：

```text
blue
is
sky
the
```

這剛好就是輸出順序。

---

## 7. 需要的知識點

這題主要需要理解：

```text
1. string 字串索引
2. while 迴圈
3. 從右往左掃描
4. 如何跳過多餘空格
5. 如何找出一個單字的左右邊界
6. substr() 擷取子字串
7. 如何控制輸出時只放一個空格
```

---

## 8. LeetCode 可直接提交程式碼

```cpp
class Solution {
public:
    string reverseWords(string s) {
        string ans;
        int i = s.size() - 1;

        while (i >= 0) {
            // 1. 跳過空格
            if (i >= 0 && s[i] == ' ') {
                i--;
                continue;
            }

            // 如果已經沒有字元，結束
            else if (i < 0) {
                break;
            }

            // 2. 如果 ans 不是空的，先加一個空格
            if (!ans.empty()) {
                ans += ' ';
            }

            // 3. 記錄單字結尾
            int end = i;

            // 4. 找到單字開頭
            while (i >= 0 && s[i] != ' ') {
                i--;
            }

            // 5. 加入目前單字
            ans += s.substr(i + 1, end - i);
        }

        return ans;
    }
};
```

---

## 9. 程式流程總覽

這支程式的主要流程可以整理成：

```text
1. 從字串最後一個字元開始往前掃描
2. 遇到空格就跳過
3. 遇到單字時：
   - 記錄單字結尾
   - 往左找單字開頭
   - 用 substr 擷取這個單字
   - 接到答案 ans 後面
4. 若 ans 已經有內容，加入下一個單字前先補一個空格
5. 重複直到整個字串掃描完
```

## 11. 程式逐行說明

---

### 11.1 從字串最後一格開始

```cpp
int i = s.size() - 1;
```

例如：

```text
s = "the sky is blue"
```

一開始：

```text
i 指向最後一個字元 'e'
```

---

### 11.2 主迴圈

```cpp
while (i >= 0) {
```

只要還沒有掃到字串最前面之外，就繼續處理。

---

### 11.3 跳過空格

```cpp
if (i >= 0 && s[i] == ' ') {
    i--;
    continue;
}
```

如果目前 `s[i]` 是空格：

```text
直接往左移一格
並跳回 while 開頭
```

---

#### 為什麼要 `continue`？

因為空格不是單字的一部分，不需要往下做：

```text
記錄 end
找單字開頭
substr 擷取
```

所以直接略過即可。

---

### 11.4 `else if (i < 0)` 的補充

```cpp
else if (i < 0) {
    break;
}
```

這段在目前程式中其實**不會真正執行到**。

原因是外層條件已經寫成：

```cpp
while (i >= 0)
```

只要進得來 while，當下就代表：

```text
i 一定 >= 0
```

所以：

```cpp
else if (i < 0)
```

在這個版本裡屬於**多餘但不影響結果**的判斷。

---

### 可以簡化成

```cpp
if (s[i] == ' ') {
    i--;
    continue;
}
```

不過保留原寫法也能正確執行。

---

### 11.5 如果 ans 已有內容，就先補空格

```cpp
if (!ans.empty()) {
    ans += ' ';
}
```

這段用來確保：

```text
單字之間只有一個空格
```

---

#### 為什麼不是每次都先加空格？

因為第一個加入的單字前面不能有空格。

---

### 範例

第一次加入：

```text
ans = "blue"
```

不用先加空格。

第二次加入 `"is"` 前：

```text
ans = "blue "
```

再變成：

```text
ans = "blue is"
```

---

### 11.6 記錄單字結尾

```cpp
int end = i;
```

例如目前掃到：

```text
blue
```

而 `i` 指向：

```text
'e'
```

那麼：

```text
end 就是 blue 的最後位置
```

---

### 11.7 找出單字開頭

```cpp
while (i >= 0 && s[i] != ' ') {
    i--;
}
```

這段會讓 `i` 持續往左移動，直到：

```text
遇到空格
或
跑到字串最左邊外面
```

---

#### 範例

```text
s = "the sky is blue"
```

處理 `"blue"` 時：

```text
i 一開始指向 'e'
```

往左走後：

```text
e -> u -> l -> b
```

最後停在：

```text
blue 前面的空格
```

---

### 11.8 用 substr 擷取單字

```cpp
ans += s.substr(i + 1, end - i);
```

---

## 12. `substr(i + 1, end - i)` 為什麼這樣寫？

### `substr(start, length)`

```cpp
s.substr(起始位置, 長度)
```

---

### 這裡：

```cpp
s.substr(i + 1, end - i)
```

代表：

```text
從單字真正開始的位置 i + 1 開始
取出 end - i 個字元
```

---

## 13. 範例解析：擷取 `"blue"`

假設：

```text
s = "the sky is blue"
```

當處理 `"blue"` 時：

```text
i   = blue 前面的空格位置
end = 'e' 的位置
```

可以畫成：

```text
the sky is blue
          ↑   ↑
          i  end
```

---

### 單字開始位置

```text
i + 1
```

就是：

```text
'b'
```

---

### 單字長度

```text
end - i
```

剛好就是：

```text
4
```

所以：

```cpp
s.substr(i + 1, end - i)
```

會取得：

```text
"blue"
```

## 14. 完整範例走一次

### 輸入

```text
s = "  hello   world  "
```

---

### 初始

```text
ans = ""
i 指向最後一個空格
```

---

### 第 1 階段：跳過尾端空格

程式會透過：

```cpp
if (s[i] == ' ') {
    i--;
    continue;
}
```

一路跳過最後面的空格。

---

### 第 2 階段：找到 `"world"`

找到字元：

```text
'd'
```

記錄：

```text
end = 'd' 的位置
```

接著往左找到 `"world"` 前面的空格。

擷取：

```text
"world"
```

加入：

```text
ans = "world"
```

---

### 第 3 階段：跳過中間多個空格

程式再次進入：

```cpp
if (s[i] == ' ') {
    i--;
    continue;
}
```

會把 `"world"` 與 `"hello"` 中間的多個空格全部跳過。

---

### 第 4 階段：找到 `"hello"`

擷取：

```text
"hello"
```

因為：

```text
ans 不為空
```

所以先加入一個空格，再接上 `"hello"`：

```text
ans = "world hello"
```

---

### 最終回傳

```text
"world hello"
```

## 15. 為什麼這個方法能自動移除多餘空格？

因為程式只會在兩種情況下加空格：

```cpp
if (!ans.empty()) {
    ans += ' ';
}
```

也就是：

```text
只在兩個輸出單字之間加一個空格
```

---

### 原始字串中的：

```text
前導空格
結尾空格
多重空格
```

都會在這段被跳過：

```cpp
if (s[i] == ' ') {
    i--;
    continue;
}
```

因此輸出格式會自動正規化。

## 16. 複雜度分析

假設：

```text
n = s.length()
```

### 時間複雜度

```text
O(n)
```

原因是：

```text
每個字元最多被主流程掃描一次
```

此外：

```cpp
substr()
```

會複製單字內容到答案中，整體輸出字元數也最多是 `n`，所以總體仍可視為 `O(n)`。

---

### 空間複雜度

```text
O(n)
```

因為需要建立新的答案字串：

```cpp
string ans;
```

## 18. 這個版本可以再精簡的地方

目前程式中的：

```cpp
else if (i < 0) {
    break;
}
```

在外層：

```cpp
while (i >= 0)
```

條件下不會被執行，因此可以刪除。

---

### 精簡後版本

```cpp
class Solution {
public:
    string reverseWords(string s) {
        string ans;
        int i = s.size() - 1;

        while (i >= 0) {
            // 跳過空格
            if (s[i] == ' ') {
                i--;
                continue;
            }

            // 如果 ans 不是空的，先加一個空格
            if (!ans.empty()) {
                ans += ' ';
            }

            // 記錄單字結尾
            int end = i;

            // 找到單字開頭
            while (i >= 0 && s[i] != ' ') {
                i--;
            }

            // 加入目前單字
            ans += s.substr(i + 1, end - i);
        }

        return ans;
    }
};
```

## 19. 面試講法

### 中文版

我從字串尾端開始往前掃描。  
如果遇到空格，就直接跳過。  
如果遇到單字，先記錄這個單字的結尾位置，再往左找到它的開頭位置，接著用 `substr()` 擷取該單字並加入答案中。  
為了確保輸出中單字之間只有一個空格，我只在 `ans` 不為空時才先加入空格。  
這樣可以同時完成反轉單字順序與移除多餘空白。

---

### English Version

I scan the string from right to left.  
If I encounter spaces, I skip them directly.  
When I find a word, I record its ending position, move left until I reach its beginning, and then use `substr()` to append the word to the result.  
To ensure that the output contains exactly one space between words, I only add a space when the result string is not empty.  
This allows me to reverse the word order while removing unnecessary spaces.
