## 1. 題目說明

這題給你兩個非空的 linked list，分別代表兩個非負整數。

每個節點只存放一個數字 digit，範圍是：

```text
0 <= Node.val <= 9
```

而且數字是**反向儲存**的。

---

## 2. 什麼叫反向儲存？

假設 linked list 是：

```text
l1 = [2, 4, 3]
```

它不是代表 `243`，而是代表：

```text
342
```

因為題目說 digits are stored in reverse order。

也就是：

```text
2 -> 4 -> 3
個位 十位 百位
```

所以：

```text
[2, 4, 3] = 342
```

---

## 3. 題目範例

### Example 1

```text
Input:
l1 = [2,4,3]
l2 = [5,6,4]

Output:
[7,0,8]
```

因為：

```text
l1 = [2,4,3] = 342
l2 = [5,6,4] = 465
```

相加：

```text
342 + 465 = 807
```

所以輸出：

```text
[7,0,8]
```

因為 `807` 反向儲存就是：

```text
7 -> 0 -> 8
```

---

## 4. 這題要解決什麼？

這題不是要把 linked list 轉成整數再相加。

原因是 linked list 最多可以有 100 個節點，也就是數字最多可能有 100 位數。

這會遠遠超過 `int` 或 `long long` 的範圍。

所以正確方法是：

```text
像手算加法一樣，一位一位加，並處理進位 carry。
```

---

## 5. 需要的知識點

### 5.1 Linked List

LeetCode 中的 linked list 節點通常長這樣：

```cpp
struct ListNode {
    int val;
    ListNode *next;

    ListNode() : val(0), next(nullptr) {}
    ListNode(int x) : val(x), next(nullptr) {}
    ListNode(int x, ListNode *next) : val(x), next(next) {}
};
```

每個節點有兩個部分：

```text
val  : 目前節點的數字
next : 指向下一個節點
```

例如：

```text
2 -> 4 -> 3
```

代表：

```text
第一個節點 val = 2
第二個節點 val = 4
第三個節點 val = 3
```

---

### 5.2 Pointer 指標

在這題中：

```cpp
ListNode* l1
ListNode* l2
```

代表 `l1` 和 `l2` 都是指向 linked list 節點的指標。

如果要取得目前節點的值：

```cpp
l1->val
```

如果要移動到下一個節點：

```cpp
l1 = l1->next;
```

---

### 5.3 Carry 進位

這題最重要的概念是進位。

例如：

```text
8 + 7 = 15
```

目前位數只能存一個 digit，所以：

```text
digit = 15 % 10 = 5
carry = 15 / 10 = 1
```

也就是：

```text
這一位放 5
下一位多加 1
```

---

### 5.4 Dummy Node 虛擬頭節點

程式中使用：

```cpp
ListNode dummy(0);
ListNode* cur = &dummy;
```

`dummy` 是一個虛擬頭節點。

它的用途是：

```text
幫助我們更方便地建立答案 linked list。
```

如果不用 dummy node，就要額外處理第一個節點，例如：

```text
如果 head 是空的，要先建立 head
如果 head 已經存在，才接到後面
```

使用 dummy node 後，就可以統一寫成：

```cpp
cur->next = new ListNode(digit);
cur = cur->next;
```

最後答案從：

```cpp
dummy.next
```

開始回傳。
