## 1. 題目目標

給定一個整數陣列 `nums` 和一個目標值 `target`，請找出陣列中兩個數字的 index，使得這兩個數字相加等於 `target`。

題目保證：

- 一定存在一組答案
- 不能使用同一個元素兩次
- 回傳順序不限

---

## 2. 題目範例

### Example 1

```text
Input: nums = [2, 7, 11, 15], target = 9
Output: [0, 1]
```

因為：

```text
nums[0] + nums[1] = 2 + 7 = 9
```

所以回傳：

```text
[0, 1]
```

---

## 3. 先理解：Hash Table 是什麼？

Hash Table 是一種可以用 **key 快速找到 value** 的資料結構。

可以把它想成一張查詢表：

```text
key  -> value
```

例如：

```text
姓名 -> 電話
學號 -> 成績
數字 -> index
```

在 Two Sum 這題中，我們會使用 Hash Table 來記錄：

```text
數字 -> 這個數字出現的位置 index
```

例如：

```text
2 -> 0
7 -> 1
11 -> 2
15 -> 3
```

代表：

```text
數字 2 出現在 index 0
數字 7 出現在 index 1
數字 11 出現在 index 2
數字 15 出現在 index 3
```

---

## 4. Hash Table 為什麼快？

假設我們想知道數字 `7` 是否出現在陣列中。

如果用一般陣列搜尋，可能要從頭開始找：

```cpp
for (int i = 0; i < nums.size(); i++) {
    if (nums[i] == 7) {
        return i;
    }
}
```

這種方法的時間複雜度是：

```text
O(n)
```

因為最壞情況下要看完整個陣列。

但是 Hash Table 可以透過 key 快速找到 value，平均查找時間是：

```text
O(1)
```

也就是幾乎可以直接查到。
